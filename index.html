ListBoost

An Android library that will implement these major features:

ListView and ExpandableListView

  • Sliding list item menus
  • Swipe-to-reveal menus
  • Drag to reorder
  • MultiSelect mode

ChangeLog

ListBoost v0.1 (Current)

  • ListView sliding list item menus is working for both cursor and non-cursor adapters. fixme: saveInstanceState isn't functioning
  • ExpandableList multiSelect mode is mostly scaffolded out.
  • Demo app is functional with sliding menu functionality for BoostListView implemented.
