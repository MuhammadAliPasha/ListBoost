Listboost

An Android library that implements sliding list item menus, swipe to reveal menus, and other list tools and functionality for both ListView and ExpandableListView

Download .zip Download .tar.gz View on GitHub

ListView and ExpandableListView

  • Sliding list item menus
  • Swipe-to-reveal menus
  • Drag to reorder
  • MultiSelect mode
